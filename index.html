Featured Projects

Real-world systems I've designed and built — each tackling complex engineering challenges at the intersection of backend, cloud, and AI.

Featured

School Bus Monitoring System

An end-to-end real-time facial recognition system for monitoring student attendance on school buses. Features a multi-threaded video processing pipeline with OpenCV + dlib for face detection/encoding, a Spring Boot REST API for backend logic, and a PostgreSQL database for attendance records. Handles concurrent camera streams with O(n log n) matching against enrolled faces.

Python OpenCV Spring Boot PostgreSQL Docker
AI / NLP

AI-Powered PDF RAG System

A Retrieval-Augmented Generation pipeline that lets users upload PDF documents and ask natural language questions. Built with a FastAPI backend, documents are chunked, embedded via sentence-transformers, and stored in ChromaDB for sub-second similarity retrieval. Integrated the Google Gemini API as the generative backbone with prompt engineering techniques for context-grounded answers — reducing hallucinations by ~70% while achieving p95 query latency under 2 seconds.

Python FastAPI LangChain Gemini API ChromaDB Docker
Computer Vision

Document OCR Tool

A high-accuracy Optical Character Recognition tool engineered to extract structured text from complex document layouts — including multi-column PDFs, scanned images, and handwritten notes. Combines Tesseract OCR with custom preprocessing pipelines (deskew, noise reduction, adaptive thresholding) and a Node.js backend for batch processing. Outputs structured JSON with bounding-box coordinates and confidence scores.

Python Tesseract Node.js OpenCV AWS S3
